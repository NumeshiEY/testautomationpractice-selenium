# testautomationpractice-selenium

Selenium TestNG automation framework for  
https://testautomationpractice.blogspot.com/

---

## 📌 Project Overview

This project is created to automate the following website:

➡️ https://testautomationpractice.blogspot.com/

The goal of this project is to practice and demonstrate:

- Selenium WebDriver automation
- TestNG framework usage
- Page Object Model (POM) design
- Git & GitHub version control best practices

---

## 🛠 Tech Stack

- **Language:** Java  
- **Automation Tool:** Selenium WebDriver  
- **Test Framework:** TestNG  
- **Build Tool:** Maven  
- **IDE:** IntelliJ IDEA  
- **Version Control:** Git  
- **Repository:** GitHub
